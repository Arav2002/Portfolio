Industrial Project

The solution is equipped with robust role-based security and multi-level access control, providing a secure and scalable platform.

Timesheet Extraction and Payroll Generation System

Industrial Project
A payroll automation system leveraging Azure AI Document Intelligence, DevExpress Blazor, C#, and PostgreSQL to streamline timesheet extraction and payroll generation. This 2-tier architecture ensures accurate payroll calculations, real-time data insights, and robust validation mechanisms to minimize errors, enhancing efficiency and reliability in salary processing.

CivilXpress

Industrial Project
The CivilXpress project centered on implementing and customizing ERP modules. With a focus on sales accounting, it seamlessly integrated three custom modules—progressive claims, sales estimation, and project budgeting—featuring multi-stage verification and customizable XML reports. IoT-Based Machine Learning Model for Biogas Production Monitoring

Academic Project
Developed an IoT-based machine learning model to monitor and optimize biogas production by integrating real-time data from pH, ammonia, methane, temperature, and pressure sensors.
